# zero player meta relational least principle action

https://github.com/user-attachments/assets/a6f966c4-aa4d-4207-9611-0a76adba3a1b

a zero player game using meta relations and least principle actions.

to compile and run:

1. clone repo

2. nvcc -O3 zero-player-meta-relational-least-principle-action.cu -lglfw -lGL -lGLEW -o zero-player-meta-relational-least-principle-action

3. ./zero-player-meta-relational-least-principle-action.cu

video is in 480p and has a 15s duration due to github's constrains of 10mb sized videos.
