a zero player game using meta relations and least principle actions.

to compile and run:

1. clone repo (git clone git@github.com:mmtmn/zero-player-meta-relational-universe.git)

2. compile it (nvcc -O3 zero-player-meta-relational-least-principle-action.cu -lglfw -lGL -lGLEW -o zero-player-meta-relational-least-principle-action)

3. run it (./zero-player-meta-relational-least-principle-action)
